HIT Instructions
You will be shown a search query and a web page. Assume that you were to issue the given query to a search engine (for example Google) and were shown the corresponding webpage as search result, answer 5 questions regarding the page. These questions assess the following aspects:
  • Relevance: The given webpage is relevant if it provides information requested in the query. For example, if the query is 'president of USA' and the page is about 'Barak Obama', it is considered relevant, on the other hand if the page is about 'Cotton candy', it is irrelevant. You shall only be payed if you mark text that contains the answer to the query.
  • Findability: We also ask you to mention how easy was it to find the information required to answer the query from the webpage. We also ask you about the time it took to find required information..

  • Instructions: If you forget instructions, please click on 'Instructions' button to view them.
  • Important: For each page please mark paragraphs that contain answer to the query using the 'Important' button.
  • View: To view the portions you marked important, you can click 'View' button.
